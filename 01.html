1. HTML 개요

HTML ; Hyper Text Markup Language
웹페이지를 만드는 언어


※ 웹페이지에서 저장된 html 문서를 열 때 : ctrl+o

2. TAG 종류

※ TAG : 웹페이지 작성을 위한 문법, 꺽새(< ~ />)를 통해 구분한다.

    1. 작성 관련 태그
  • title : 웹페이지의 제목을 설정
  • meta charset = "문자열 형식": 브라우저가 읽을 html문서 문자형식 지정(character set)

  • h1,h2,h3,h4,h5,h6 : 6단계로 제목을 구분하는 태그 (글자크기, 굵기, 줄바꿈)
  • br : 줄바꿈
  • p : 단락 구분(앞 뒤, 줄바꿈)

  • strong : 진하게
  • u : 밑줄
  • img src="이미지파일명" width="화면비율" : 이미지 삽입
    • ※ alt 누르고 마우스 왼쪽 클릭 : 다중 선택

    2. 구조 관련 태그
  • li : 점을 통해 리스트형식으로 전환
  • ul : 하위단계의 리스트들을 묶어줌(unordered list)
  • ol : 하위단계의 리스트들을 순서대로 넘버링 하며 묶어줌(ordered list)

  • head : 본문을 설명하는 상위 태그
  • body : 본문을 직접 나타내는 상위 태그
  • html : head, body태그를 포괄하는 html 최상위 태그

  • a href = "웹페이지 주소" : 다른 웹페이지를 연결하는 태그(ankor hypertext reference)
  • a href = " " target="_blank" : 새 탭에서 다른 웹페이지를 열기
  • a href = " " title = '설명문구' : 마우스를 올려놓았을 때 설명문구를 표기
